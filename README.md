# UNES - Página Principal

Este projeto consiste em um site estático simples para a **UNES - Universidade**, desenvolvido com HTML básico. Ele apresenta a página principal da universidade, informações sobre a instituição e uma página de contato.

## Funcionalidades

*   **Página Inicial (index.html):** Apresenta a universidade com informações gerais e imagens.
*   **Quem Somos (quemsomos.html):** Detalha a missão, visão e valores da instituição.
*   **Contato (contato.html):** Oferece informações de contato.
*   **Navegação:** Links entre as páginas para facilitar a exploração do site.
*   **Imagens:** Utilização de imagens para enriquecer o design visual.

## Tecnologias Utilizadas

*   **HTML:** Para a estruturação e conteúdo das páginas web.

## Como Usar

Para visualizar o site, basta abrir o arquivo `index.html` em qualquer navegador web. Certifique-se de que a estrutura de pastas (`imagens/`) esteja intacta para que as imagens sejam carregadas corretamente.
